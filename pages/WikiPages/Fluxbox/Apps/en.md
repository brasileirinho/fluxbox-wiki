# Fluxbox Menu System

**Man-Page:**  
``:~/$ man fluxbox-apps``
{.alert-info .alert}

## Default Menus
There are 3 Menus in fluxbox by default.
- Main Application Menu, right click on the desktop
- Window Menu, right click on a windows titlebar
- Workspace Menu, middle click on the desktop
